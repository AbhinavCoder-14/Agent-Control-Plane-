# CONTEXT

Project idea: Build an ecommerce website with:
- product listing
- cart
- checkout (mock)
- admin dashboard

## Product Goals
- Let users browse products and view basic details.
- Let users add, update, and remove cart items.
- Let users complete a mock checkout with order confirmation.
- Let admins manage products and monitor mock orders.

## Scope For Agent

- Prefer deterministic, maintainable code.
